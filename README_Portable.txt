WHAT IS ORCAVIEW?
ORCAView is a modern graphical user interface (GUI) for the ORCA Quantum 
Chemistry Program. It simplifies the process of creating, running, and 
analyzing ORCA calculations with a professional, feature-rich interface.

SYSTEM REQUIREMENTS:
- Windows 10/11 (64-bit)
- 2GB RAM minimum (4GB recommended)
- 1GB free disk space
- ORCA Quantum Chemistry Program (separate download)

INSTALLATION INSTRUCTIONS:
1. Extract the entire ORCAView folder to your desired location
2. Double-click "ORCAView.exe" to launch the application
3. No additional software installation required!

FIRST TIME SETUP:
1. Launch ORCAView
2. Go to the "Submission" tab
3. Click "Browse..." next to "ORCA Executable Path"
4. Navigate to your ORCA installation and select "orca.exe"
5. The path will be saved automatically for future use

KEY FEATURES:
✓ Comprehensive method support (80+ DFT functionals, 100+ basis sets)
✓ All ORCA 6.1 compatible semiempirical methods and xTB methods
✓ Interactive 3D molecule viewer with trackball controls
✓ 2D molecular structure depiction and editing
✓ Job queue management with real-time status monitoring
✓ Custom input block support for advanced ORCA features
✓ Ketcher molecular editor integration
✓ SMILES to 3D structure generation
✓ XYZ file import/export capabilities

GETTING ORCA:
ORCAView requires the ORCA Quantum Chemistry Program to perform calculations.
Download ORCA from: https://orcaforum.kofo.mpg.de/

TROUBLESHOOTING:
- If the application doesn't start, ensure you have Windows 10/11 64-bit
- If ORCA jobs fail, verify the ORCA executable path in Settings
- For 3D viewer issues, update your graphics drivers
- Check Windows Defender/antivirus if the .exe is blocked
